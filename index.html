Quick Start Guide

  1. Set Up Your Checklist: Visit the Checklist page to create tiers and add businesses. Use the Configuration Management panel to organize your businesses by tier with custom icons and colors.
  2. Track Your Progress: Use the Checklist to monitor money, stock levels (with automatic calculations), collection storage, and notes for each business. All data is saved automatically to your browser.
  3. Track Vehicle Progress: Import and track your vehicle delivery progress, monitor reputation requirements, and see unlock status in the Vehicle Deliveries section. Each page includes a collapsible how-to guide with step-by-step instructions.
  4. Monitor Merchants: Use the Traveling Merchants page to track merchant rotations, locations, prices, and timers. Import data using the /merchants command in-game.
  5. Track Training: Monitor education timers and training progress with the Education Timer tool. Import training data from the University of Los Santos.
  6. Use Debug Mode: Enable debug mode on any page to see detailed logging and troubleshoot data parsing issues.
  7. Export Your Data: Use the export features in the Checklist to backup your configuration or generate CSV reports.

About The Tycoon

The Tycoon is a business simulation game where players build and manage various businesses, from simple fruit stands to complex fisheries. The game challenges players to optimize their operations, manage resources, train employees, and grow their business empire.

This website provides comprehensive tools and resources to help you succeed in The Tycoon. Features include:

  • Data Import: Import data directly from the game using in-game commands (/merchants, vehicle dealer data, training queue data, etc.)
  • Local Storage: All your progress is saved locally in your browser for quick access - no server or database required
  • Real-Time Timers: Track merchant rotations, education timers, and server reboots with live countdowns
  • Local Time Display: All times are displayed in your local timezone for convenience
  • Tier & Business Management: Create, edit, and organize tiers and businesses with custom icons, colors, and emoji picker with search functionality
  • Stock Calculations: Automatic stock calculations showing how much stock is needed to reach maximum capacity
  • Number Formatting: Large numbers are automatically formatted with commas for better readability
  • Debug Mode: Enable debug logging on all pages to troubleshoot and understand data flow
  • How-to Guides: Collapsible step-by-step instructions included on each page with keyboard shortcuts for easy setup
  • Import/Export: Export your checklist configuration as JSON or generate CSV reports for external use
  • Client-Side Only: All tools work entirely client-side with no API or database calls, ensuring your data stays private and accessible even offline

Privacy & Security: All data is stored locally in your browser. No data is sent to any server, ensuring complete privacy and security.
