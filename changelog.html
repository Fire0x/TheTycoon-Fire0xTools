How this page works

This page now uses a modular system for managing changelog entries:

  1. Recent Entries: Keep the latest 2-3 versions in changelog.html for immediate visibility.
  2. Archive Files: Older entries are stored in pages/changelog/ directory as separate HTML files (e.g., changelog-archive-1.html).
  3. Loading Archives: Use the "Load Archive Files" section above to select and load older changelog entries.
  4. Load More: After loading archives, use the "Load More" button to progressively display additional entries.
  5. Adding New Entries: Copy an existing <article class="changelog-entry"> block and paste it ABOVE existing entries in changelog.html.
  6. Moving to Archive: When you have more than 3 recent entries, move older ones to archive files in pages/changelog/.
