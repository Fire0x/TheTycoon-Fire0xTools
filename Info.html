Paragraphs

Paragraphs are the basic building blocks of text content. Use them to write regular text, descriptions, and explanations.

This is a regular paragraph. You can write multiple sentences here. Paragraphs automatically wrap to fit the container width.

You can have multiple paragraphs in a row. Each paragraph will have spacing between them for better readability.

HTML Example:

<p>This is a paragraph with regular text content.</p>
<p>This is another paragraph.</p>

Text Formatting

You can format text to emphasize important information or highlight specific content.

This is bold text using the <strong> tag.

This is italic text using the <em> tag.

You can combine bold and italic formatting: bold and italic.

This is highlighted text using the <mark> tag.

This is smaller text using the <small> tag.

This is deleted text using the <del> tag.

This is inserted text using the <ins> tag.

This is subscript and this is superscript text.

Bullet Point Lists

Use unordered lists (bullet points) to present items that don't have a specific order or sequence.

Simple List

  • First item

  • Third item

List with Formatting

Nested List

Numbered Lists

Use ordered lists (numbered lists) when items have a specific sequence or order that matters.

Simple Numbered List

  1. First step
  2. Second step
  3. Third step

List with Formatting

  1. Step one: Do this first
  2. Step two: Then do this
  3. Step three: Finally, do this

Nested Numbered List

  1. Main step 1
    1. Sub-step 1.1
    2. Sub-step 1.2
  2. Main step 2
    1. Sub-step 2.1
    2. Sub-step 2.2

HTML Example:

<ol>
    <li>First step</li>
    <li>Second step</li>
    <li>Third step</li>
</ol>

Code Blocks

Use code blocks to display code snippets, commands, or technical information.

Inline Code

Use <code> for inline code snippets like function names or commands: console.log() or /merchants.

Code Blocks

Use <pre><code> for multi-line code blocks:

function example() {
    console.log("Hello, World!");
    return true;
}

Command Examples

/merchants
/checklist
/vehicles

HTML Example:

<!-- Inline code -->
<p>Use <code>console.log()</code> to print.</p>

<!-- Code block -->
<pre><code>function example() {
    return true;
}</code></pre>

Blockquotes

Use blockquotes to highlight quotes, important notes, or callouts.

This is a blockquote. Use it to highlight important information, quotes, or special notes.

Optional citation or source

You can have multiple blockquotes on a page. Each one will be visually distinct.

HTML Example:

<blockquote class="blockquote">
    <p>This is a quote or important note.</p>

</blockquote>
